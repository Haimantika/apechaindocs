# Oracles

## Pyth

[Pyth](https://www.pyth.network/) acts as an oracle for ApeChain by providing real-time, reliable data feeds directly to the blockchain. Pyth delivers real-time, reliable data which is vital for Web3 dApps, especially in gaming and DeFi, where smart contracts rely on accurate data. With Pyth, ApeChain gives developers access to high-quality data, boosting the performance of projects built on Apechain.
